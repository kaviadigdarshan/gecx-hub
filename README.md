# GECX Accelerator Hub

## Productivity tools for Google CX Agent Studio (ces.googleapis.com)

- Browse, search, and manage Agent Studio resources (agents, flows, intents, entity types) across projects
- Bulk import and export conversation flows in human-readable formats
- Validate agent configurations and surface common misconfigurations
- Compare agent versions and generate diff reports
- Generate boilerplate intent and fulfillment code from templates
- Trigger test conversations and view session transcripts
